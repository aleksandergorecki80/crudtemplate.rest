# crudtemplate.rest

> Backend API for CRUD template application with posts, comments and users. Admin mode allows users administration.

## Usage

Rename "dev.json.json" to "dev.json" and update settings to your own.

## Instal Dependencies

```
npm install
```

## Run App

- Run in dev mode

```
npm run dev
```

- Run in prod mode

```
npm statr
```
